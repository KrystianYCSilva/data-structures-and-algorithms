# Domain Concepts

Key domain concepts for the algorithms and optimization library.

- Algorithm: A step-by-step procedure to perform computations; prefer pure functions for deterministic behavior.
- Data Structure: Collections and organization of data (Arrays, Lists, Trees, Graphs) optimized for operations required by algorithms.
- Invariant: Conditions that must hold true across function boundaries (use unit/property tests to assert invariants).
- Complexity: Big O notation for time and space — record expected complexities in documentation for each algorithm.
