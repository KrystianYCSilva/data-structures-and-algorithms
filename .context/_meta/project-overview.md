# Project Overview

## What

A Kotlin Multiplatform library of data structures and optimization algorithms for educational and research purposes.

## Why

Fornecer implementações portáveis de algoritmos fundamentais e otimização que funcionem across JVM, JS and native targets, suitable for study, benchmarks and reuse.

## Scope

Includes core data structures, algorithm implementations, and KMP-friendly utilities; excludes full application services or production-grade deployment scripts.

## Status

Prototype / library stage. Actively developed with tests and KMP targets configured.
